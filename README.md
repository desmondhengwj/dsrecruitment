# DS Recruitment

## Heroku Link
https://dsrecruitment.herokuapp.com/

## Running the app

You can run the app with the following commands:

```
$ git clone https://github.com/desmondhengwj/dsrecruitment.git
$ cd dsrecruitment
$ bundle install
$ rails server
```
